# Microtastic Roadmap

The guiding principle is **zero config for pure ES6 browser development** — no bundler
configuration, no framework lock-in, just fast dev and clean prod builds.

Each release has a subfolder in `docs/` containing design documents for the planned
features (e.g. `docs/v0.0.72/`).

## Out of scope

TypeScript compilation, JSX transforms, or any framework-specific tooling. Microtastic
